# Simple LangChain Application

A simple command-line application demonstrating how to use LangChain with OpenAI to generate interesting facts based on a given topic.

## Setup

1. **Create and activate a virtual environment** (optional but recommended):
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure Environment Variables**:
   Copy the example environment variables file and add your OpenAI API key.
   ```bash
   cp .env.example .env
   ```

## Usage

Run the application:
```bash
python app.py
```
